Superstore Profit Analyzer is an end-to-end Machine Learning project designed to predict retail profit and support data-driven business decisions.

The project covers the complete ML workflow, including:

• Data Cleaning & Preprocessing
• Exploratory Data Analysis (EDA)
• Feature Engineering
• Model Training & Evaluation
• Profit Prediction using Machine Learning
• Interactive Business Analytics Dashboard
• KPI Monitoring & Visualization
• Business Insight Generation

The trained model is deployed through a Streamlit-based web application that enables users to input business parameters and receive profit predictions along with actionable insights and visual analytics.

Tech Stack:
Python, Pandas, NumPy, Scikit-Learn, Plotly, Streamlit, Joblib
